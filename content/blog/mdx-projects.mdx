## Technical Implementation

The navigation system uses Material UI's Drawer component with custom animations:

<CodeDemo 
  title="Custom Animated Navigation" 
  description="Navigation drawer with Framer Motion animations"
  code={`
import { Drawer, List, ListItem, ListItemText } from '@mui/material';
import { motion } from 'framer-motion';

// Animated list item
const AnimatedListItem = motion(ListItem);

const Navigation = ({ open, onClose }) => {
  return (
    <Drawer
      anchor="left"
      open={open}
      onClose={onClose}
    >
      <List sx={{ width: 250 }}>
        {['Home', 'Projects', 'Blog', 'Contact'].map((text, index) => (
          <AnimatedListItem
            button
            key={text}
            initial={{ x: -20, opacity: 0 }}
            animate={{ x: 0, opacity: 1 }}
            transition={{ delay: index * 0.1 }}
            onClick={onClose}
          >
            <ListItemText primary={text} />
          </AnimatedListItem>
        ))}
      </List>
    </Drawer>
  );
};
`}
>
  <div style={{ padding: '10px' }}>
    <div style={{ marginBottom: '10px', fontWeight: 'bold' }}>Navigation Menu</div>
    <div style={{ border: '1px solid #ccc', borderRadius: '4px', overflow: 'hidden' }}>
      <div style={{ padding: '10px 15px', borderBottom: '1px solid #eee', backgroundColor: '#f0f8ff' }}>Home</div>
      <div style={{ padding: '10px 15px', borderBottom: '1px solid #eee', backgroundColor: '#f0f8ff' }}>Projects</div>
