# MongoDB Search Project

This project demonstrates the integration of MongoDB Atlas Vector Search with OpenAI for semantic search capabilities.

## Architecture

```
┌──────────────┐     ┌───────────┐     ┌──────────────┐
│              │     │           │     │              │
│  User Input  │────▶│  Next.js  │────▶│  OpenAI API  │
│              │     │           │     │              │
└──────────────┘     └───────────┘     └──────────────┘
                          │                    │
                          │                    │
                          ▼                    ▼
                    ┌───────────┐      ┌──────────────┐
                    │           │      │              │
                    │ MongoDB   │◀─────│   Vector     │
                    │ Atlas     │      │  Embeddings  │
                    │           │      │              │
                    └───────────┘      └──────────────┘
                          │
                          │
                          ▼
                    ┌───────────┐
                    │           │
                    │  Search   │
                    │  Results  │
                    │           │
                    └───────────┘
```

## Features

- Semantic search using OpenAI embeddings
- Vector similarity search with MongoDB Atlas
- Real-time search results
- Scalable architecture

## Implementation Details

The project implements a sophisticated search system that:

1. Takes user input and processes it through Next.js
2. Generates embeddings using OpenAI's API
3. Stores and retrieves vectors using MongoDB Atlas
4. Returns semantically relevant search results
